# Grand Melee Slot

Grand Melee is a slot by Thunderkick released in August of 2023.

## Symbols

In Grand Melee symbols are 224x248

Symbols share base animations that let them be universally controlled by code
- `disable`
- `disableFade`
- `enable`
- `idle`
- `initialWin`
- `land`
- `winToggle`
- `winToggleFast`


**НУЖНО СДЕЛАТЬ ФИЛЬТР ДЛЯ АНИМАЦИЙ ЧТОБЫ НЕ ПОКАЗЫВАТЬ ВСЕ**

Grand Melee's key feature is **Sticky Reel**,  all symbols share animations to be used in that feature's logic
- `idleStickyReel`
- `landPreSticky`
- `stickyIdle`
- `stickyOff`
- `sickyOn`
- `activateStickyReel`
- `spinStart`

### Timings

Grand Melee symbols use following timings:
- Landing - 500ms or 15 frames
- Disable and Enable - 10 frames
- Initial Win - 30 frames
- Win Toggle - 40 frames
- Spin Start - 15 frames
- Sticky Reel Activation - 35 frames
